# Spring Batch Architecture Agent (SBA)

Design and implement production-ready Spring Batch applications through a guided 5-phase workflow.

## Available Commands

| Command | What it does |
|---------|-------------|
| `sba` | Start or continue the Spring Batch Architecture workflow |

## The 5-Phase Workflow

1. **Discovery** — gather requirements, data sources, volumes, constraints
2. **Architecture** — make Architecture Decision Records (ADRs) for tech stack choices
3. **Design** — detailed data models and step configurations
4. **Implementation** — generate production-ready Spring Batch code
5. **Review** — optimize, validate, and hand off
